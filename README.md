# 📊 Titanic EDA Report

Complete Exploratory Data Analysis on the Titanic dataset.
Built as part of my AI/ML learning journey.

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 What This Project Does
- Loads and explores Titanic dataset
- Analyses passenger demographics
- Fare and class analysis
- Survival analysis by gender, class, age group
- Correlation matrix and relationships
- 12+ charts with insights

## 📈 Key Insights
- Female survival rate 74% vs Male 19%
- 1st class survival 63% vs 3rd class 24%
- Children survived more than adults
- family_size and SibSp are highly correlated (0.89)
- Most passengers paid fare between 0-50

## 🚀 How to Run
pip install pandas matplotlib seaborn
python titanic_eda_report.py
